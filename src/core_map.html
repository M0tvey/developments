    object
    Настройки смены картинки маркера по клику.

    boolean
    true
    Менять ли картинку маркера при клике?
  • imageHref

    object
    Настройки кастомного html маркера.

    boolean
    false
    Использовать кастомный HTML маркера?

    iconImageHref
    и

    не работают.

    string
    Кастомный HTML маркера.
  • coordinates
    array
    [[[0, 0], [-15, -19], [-20, -33], [-14, -47], [0, -54], [14, -47], [20, -33], [15, -19], [0, 0]]]
    Полигон описывается в виде трехмерного массива. Массив верхнего уровня содержит контуры полигона. Первый элемента массива - это внешний контур, а остальные - внутренние.

    boolean
    false

    string

    ,

    string

    "left"

    "right"
